转为转义字符串

支持非标准JSON字符串输入，自动识别并修复格式后转义。

输入 (非标准字符串):
"{a:1,b:2}"
输出 (标准转义):
"{\"a\":1,\"b\":2}"

解析转义字符串

将转义后的 JSON 字符串还原为格式化的标准 JSON 对象。

输入:
"{\"name\": \"张三\"}"
输出:
{ "name": "张三" }
