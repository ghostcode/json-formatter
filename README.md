# JSON Formatter

一站式 JSON 处理工具，支持格式化、压缩、转义转换，能够智能修复非标准 JSON 数据。

## 功能特性

- **智能修复**: 自动处理单引号、无引号键名、尾随逗号、注释等非标准格式
- **格式化/压缩**: 标准 JSON 美化与压缩，支持排序键名、自定义缩进
- **转义转换**:
  - **转为转义字符串**: 支持非标准 JSON 字符串（如 `"{a:1}"`）自动修复并转义
  - **解析转义字符串**: 将转义字符串还原为格式化 JSON 对象
- **交互体验**: 暗色主题、实时状态反馈、键盘快捷键 (`Ctrl/Cmd + Enter`)、一键复制/下载

## 使用说明

### 格式化
输入非标准 JSON 数据，点击“格式化”或按 `Ctrl+Enter`，输出标准 JSON。

### 转义转换
1. **转为转义字符串**: 输入 `"{a:1,b:2}"` 或 `{a:1,b:2}`，输出 `"{\"a\":1,\"b\":2}"`
2. **解析转义字符串**: 输入 `"{\"name\":\"张三\"}"`，输出格式化的 JSON 对象
